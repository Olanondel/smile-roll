# Sushi Store 🍣

**Современный интернет-магазин суши** с упором на удобный UX, чистую архитектуру и реальный e-commerce flow.  
Проект создан как часть портфолио и демонстрирует навыки разработки frontend-приложений с полноценной бизнес-логикой, интеграцией с Firebase и удобным пользовательским интерфейсом.

---

## 🇷🇺 О проекте

Это pet-проект / portfolio-проект интернет-магазина суши, в котором реализован типичный пользовательский сценарий онлайн-заказа:

- просмотр каталога товаров
- фильтрация и поиск
- добавление товаров в корзину
- добавление товаров в избранное
- оформление заказа
- авторизация пользователя
- личный кабинет
- хранение пользовательских данных и заказов в Firebase

Проект создавался не как простой лендинг, а как полноценное SPA-приложение с акцентом на архитектуру, переиспользуемые компоненты, управление состоянием и удобство взаимодействия.

# 🇬🇧 About the Project

**Sushi Store** is a modern sushi e-commerce web application built as a portfolio project.
